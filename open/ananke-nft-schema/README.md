# Ananke NFT Schema

This schema defines NFT metadata for Ananke procedural generation rights.

## Key Principles

1. **NFTs represent procedure rights, not media**: The NFT stores only the seed and generation parameters
2. **Regeneration-only**: Content is regenerated on-demand from the seed
3. **Revocable**: All seeds can be revoked, disabling regeneration
4. **Version-pinned**: Engine version is recorded to ensure reproducibility
